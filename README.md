# Operating Systems Simulators

![Banner](/assets/banner.jpg) 

## Current Version Includes:

- Simulation of single process (simulator1.c)
- Simulation of multiple processes (simulator2.c)
- Simulation of process as threads (single) (simulator3.c)
- Parallel process simulators (simulator4.c)
- Process table, I/O simulation and hard-drive access (FCFS) (simulator5.c)
- Hard-drive access, look-scan (simulator6.c)
- Generic I/O device (single) (simulator7.c)
- Generic I/O devices (multiple) (simulator8.c)
